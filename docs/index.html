Ключевые возможности

🎬

Встроенный видеоплеер

Смотрите аниме прямо на страницах Shikimori без необходимости переходить на другие сайты

🔄

Множество источников

Поддержка плееров Turbo, Lumex, Alloha и Kodik с автоматическим переключением при сбое

⚡

Ленивая загрузка

Плеер загружается только при появлении в видимой области экрана для оптимизации производительности

🔔

Современные уведомления

Стилизованные уведомления для отображения ошибок, предупреждений и информации о статусе плеера

📱

Адаптивный дизайн

Полностью адаптируется под разные размеры экранов для комфортного просмотра на любом устройстве

💾

Кэширование

Сохраняет ответы API для уменьшения количества запросов и ускорения загрузки плеера

Поддерживаемые плееры

T
Turbo
Активен
Основной источник с высокой скоростью загрузки и хорошим качеством видео. Автоматически используется по умолчанию.
L
Lumex
Активен
Резервный источник с хорошей стабильностью и качеством. Автоматически переключается при недоступности Turbo.
A
Alloha
Активен
Дополнительный источник с большим количеством контента. Используется как третий вариант при недоступности предыдущих.
K
Kodik
Активен
Финальный резервный источник с хорошим качеством и стабильностью. Гарантирует доступность контента.

Установка

1

Установите менеджер пользовательских скриптов

Установите Tampermonkey (Chrome/Firefox/Edge) или Greasemonkey (Firefox) из официального магазина расширений вашего браузера

2

Установите ShikiPlayer

Нажмите кнопку "Установить ShikiPlayer" ниже и подтвердите установку в менеджере скриптов

3

Перейдите на Shikimori

Откройте любую страницу аниме на Shikimori (например, Hunter x Hunter (2011)) и наслаждайтесь просмотром!
